# Hedwig.pub

作者: @zwithz

举例: https://rsshub.app/blogs/hedwig/zmd

路由: /blogs/hedwig/:type

参数:

  • type, 必选 -

    分类, 见下表

呆唯的 Newsletter 0neSe7en 的技术周刊 地心引力 宪学宪卖 Comeet 每周精选 无鸡之谈 我有一片芝麻地
hirasawayui se7en walnut themez comeet sunskyxh zmd

原则上只要是 {type}.hedwig.pub 都可以匹配。
